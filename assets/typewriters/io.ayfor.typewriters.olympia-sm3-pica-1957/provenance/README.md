# Olympia SM3 Pica demo provenance

This package proves the typewriter-package plumbing; it is not a museum-grade
reconstruction.

- Historical target: Olympia SM3, 1957, Pica variant.
- Typeface: Cutive Mono Regular by the Cutive Project Authors, SIL OFL 1.1.
  It is the exact `ofl/cutivemono/CutiveMono-Regular.ttf` file from
  `google/fonts` commit `7c38f10e1cbdc02df83ef5919d9639dcf1e3474f`,
  SHA-256
  `96a36a00079058684982f61ee334323f8b501d7b68dcecd6049a4f9177e3a62c`.
- Glyph calibration: hand-authored demonstration offsets, not measurements
  from a serial-numbered specimen.
- Hammer sounds: five processed excerpts of `typewriter-olivetti-lettra-22`
  by keithpeter (Freesound), downloaded from Pixabay on 2026-07-02. They are
  not Olympia recordings; exact source, licence and processing details are in
  `licenses/sounds.txt`.

Its `fidelity` remains `inspired` until a real SM3 calibration sheet and
isolated SM3 recordings replace the demonstration data.

The Olympia name is used solely to identify the historical machine that
inspired this unofficial package. No affiliation with or endorsement by the
trademark owner is claimed.
