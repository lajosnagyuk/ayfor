# Ayfor Classic provenance

This is the immutable package form of ayfor's original built-in machine.

- Typeface: Courier Prime Regular, SIL Open Font License 1.1. It is the exact
  `fonts/ttf/CourierPrime-Regular.ttf` file from
  `quoteunquoteapps/CourierPrime` commit
  `33d9d2ca2f56e6d348f6b84d4d10a34550f74220`, SHA-256
  `72f793376f8e2841656bf21d77a5de010f2929bd6956a22ee848ad0c7eb978af`.
- Mechanics: ayfor personality model v1.
- Hammer sounds: five processed excerpts of `typewriter-olivetti-lettra-22`
  by keithpeter (Freesound), downloaded from Pixabay on 2026-07-02. Exact
  source, licence and processing details are in `licenses/sounds.txt`.

The package is called Ayfor Classic rather than Olivetti because its visual
model and typeface are ayfor's original generic machine; only the recordings
come from the named Olivetti specimen.
