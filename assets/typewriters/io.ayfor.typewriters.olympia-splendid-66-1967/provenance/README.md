# Olympia Splendid 66 Elite demo provenance

This package proves that ayfor can ship another machine with its own typeface,
12 CPI escapement, strike calibration, and sound tuning. It is not a
museum-grade reconstruction.

- Historical target: an Olympia Splendid 66 specimen dated 1967, interpreted
  as an Elite-style 12 CPI portable.
- Historical reference: https://typewriterdatabase.com/1967-olympia-splendid-66.3967.typewriter
- Typeface: Special Elite, copyright (c) 2010 by Brian J. Bonislawsky DBA
  Astigmatic (AOETI), Apache License 2.0. It is the exact
  `apache/specialelite/SpecialElite-Regular.ttf` file from
  `google/fonts` commit `0c42307921ce94c085ce191020cde436f8396ec3`,
  SHA-256
  `a776fcb4ceb8bdf03e2967688ebdad42680de5b91a7e62c17e718ae212d14bc4`.
- Glyph calibration and geometry: hand-authored engineering-demo values, not
  measurements from a serial-numbered specimen.
- Hammer sounds: five processed excerpts of `typewriter-olivetti-lettra-22`
  by keithpeter (Freesound), downloaded from Pixabay on 2026-07-02. They are
  not Olympia recordings; exact source, licence and processing details are in
  `licenses/sounds.txt`.

Splendid 66 machines could be ordered or encountered with different type and
keyboard configurations. This release therefore remains `fidelity: inspired`.
It must receive newly measured slugs, escapement, and isolated recordings—and
a new semantic version—before claiming specimen fidelity.

The Olympia name is used solely to identify the historical machine that
inspired this unofficial package. No affiliation with or endorsement by the
trademark owner is claimed.
